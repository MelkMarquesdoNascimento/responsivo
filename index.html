Layout Responsivo com Flex Box



Primeiro Passo


Para começar devemos selecionar quais elementos devem ser dispostos como containers flexíveis. Para fazer isso, definimos um valor especial de exibição no elemento pai, o container, dos elementos que voçê deseja afetar.


Segundo Passo


O container flexível vai ser estilizado . Assim, dentro do identificador que representa, vamos inserir as regras CSS responsáveis por ajustar o visual dos elementos que ficaram dentro dele.


Terceiro Passo


Esse passo e dedicado em configurar os elementos que estão dentro do contaiener flexível, isto é, seus filhos.


Apesar de todos terem feito o tratamento feito pelo container flexível, é possível fazer com que alguns deles tenham um visual que o diferencia dos demais.
